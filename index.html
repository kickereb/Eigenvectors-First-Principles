Eigenvectors from first principles

A matrix takes a vector and moves it. Most directions get stretched and rotated. But a rare few only get stretched, never rotated. Those are eigenvectors. Finding them requires building, from scratch, one of the most elegant arguments in mathematics.

To see which directions are special, we first need to see what a matrix does to an ordinary vector.

A matrix is a verb

A vector like $(3,\,2)$ is an instruction: go 3 steps along the horizontal reference arrow $\hat\imath = (1,0)$, then 2 steps along the vertical $\hat\jmath = (0,1)$. Coordinates are always an instruction written in terms of some reference pair.

A $2\times 2$ matrix is a relocation notice for those two reference arrows. Its first column is the new home of $\hat\imath$; its second column is the new home of $\hat\jmath$. For the matrix $A = \begin{pmatrix}2&1\\1&2\end{pmatrix}$, the first column $(2,1)$ tells you where $\hat\imath$ lands, and the second column $(1,2)$ tells you where $\hat\jmath$ lands.

Because the move keeps grid lines straight and evenly spaced, and since that's what linearity means, every other vector is carried along automatically. The vector that used to mean "3 steps along $\hat\imath$ and 2 steps along $\hat\jmath$" now means "3 steps along the new $\hat\imath$ and 2 steps along the new $\hat\jmath$":

$$A\mathbf{v} = x\cdot\text{(new }\hat\imath) + y\cdot\text{(new }\hat\jmath)$$

The widget below makes this tangible. The faint grid shows the whole plane being warped. Edit the matrix or choose a preset, then drag the blue arrow.

Drag enough vectors and you'll notice most get knocked off their original line. But a few don't. Those are the ones we care about.

The directions that don't turn

Load the stretch preset ($A = \begin{pmatrix}2&1\\1&2\end{pmatrix}$) and drag the blue arrow to the diagonal line through $(1,1)$. The orange output lands exactly on top — stretched to three times the length, but pointing the same way. Now drag to $(1,-1)$: the output is identical to the input, unchanged.

An eigenvector is any nonzero vector that the transformation leaves on its own line — the output is a scalar multiple of the input. Written as an equation:

$$A\mathbf{v} = \lambda\,\mathbf{v}$$

The scalar $\lambda$ is the eigenvalue — how much the transformation stretches or compresses along that direction. For the stretch matrix: the $(1,1)$ direction has $\lambda = 3$, the $(1,-1)$ direction has $\lambda = 1$. All other directions rotate off their lines; these two don't.

Reload rotate 90° and try to find an eigenvector. There are none — every direction gets turned. A matrix with no real eigenvalues is one whose transformation is an inescapable rotation. That's information, not a failure.

Fine — they're special. But given a matrix, how would we actually find them?

Finding them requires detecting a collapse

Rearrange $A\mathbf{v} = \lambda\mathbf{v}$. Subtract $\lambda\mathbf{v}$ from both sides; insert the identity matrix $I$ so both sides have a matrix on the left:

$$(A - \lambda I)\,\mathbf{v} = \mathbf{0}$$

We want a nonzero vector $\mathbf{v}$ in the kernel of $(A - \lambda I)$. That's only possible when the matrix collapses space — squashes some direction down to nothing. An invertible matrix has no such direction; the only vector it kills is zero itself. So a nonzero eigenvector exists precisely when $A - \lambda I$ is a collapsing matrix.

Collapse has one algebraic fingerprint: the determinant equals zero. The determinant measures how much a transformation scales area. A collapsing transformation scales area to zero. So the hunt for eigenvalues reduces to:

$$\det(A - \lambda I) = 0$$

This is the equation we need to solve — for the values of $\lambda$ that make it true. But to use it, we'd better understand what the determinant actually is — not just the formula, but why that formula is the only possible answer.

Everything hinges on one number that detects collapse. We'd better build it from scratch.

What the determinant actually is

Rather than memorise $ad - bc$, let's derive it. We need a function that takes a matrix's two columns and returns its "signed area scale factor". We'll write down the three properties such a function must have, and then show exactly one function satisfies all three — which forces the formula.

Think of $\det$ as a function of two column vectors. The three axioms:

  1. Multilinear: linear in each column separately. Doubling a column doubles the determinant: $\det(\alpha\mathbf{u} + \beta\mathbf{u}',\,\mathbf{v}) = \alpha\det(\mathbf{u},\mathbf{v}) + \beta\det(\mathbf{u}',\mathbf{v})$, and likewise for the second slot.
  2. Alternating: $\det(\mathbf{v},\mathbf{v}) = 0$ for any $\mathbf{v}$. A "parallelogram" whose two sides are the same vector is a degenerate line segment — no area.
  3. Normalisation: $\det(\hat\imath, \hat\jmath) = 1$. The unit square has unit area.

Now derive the formula. Write the columns of $A = \begin{pmatrix}a&b\\c&d\end{pmatrix}$ in the basis: $\mathbf{c}_1 = a\hat\imath + c\hat\jmath$ and $\mathbf{c}_2 = b\hat\imath + d\hat\jmath$. Expand by multilinearity:

$$\det(\mathbf{c}_1, \mathbf{c}_2) = ab\,\det(\hat\imath,\hat\imath) + ad\,\det(\hat\imath,\hat\jmath) + cb\,\det(\hat\jmath,\hat\imath) + cd\,\det(\hat\jmath,\hat\jmath)$$

The alternating axiom kills the first and last terms. From $\det(\hat\imath+\hat\jmath,\,\hat\imath+\hat\jmath)=0$, expanding by multilinearity gives $\det(\hat\jmath,\hat\imath) = -\det(\hat\imath,\hat\jmath)$. With normalisation, $\det(\hat\imath,\hat\jmath)=1$, so $\det(\hat\jmath,\hat\imath)=-1$. What remains:

$$\det(A) = ad\cdot 1 + cb\cdot(-1) = ad - bc$$

The formula isn't memorised — it's forced. The widget below shows the geometry: watch the unit square inflate into the parallelogram spanned by $A$'s columns. The area readout equals the determinant.

The sign carries real information. Positive: the transformation preserves orientation (counterclockwise stays counterclockwise). Negative: it flips. Zero: full collapse — the plane is squashed to a line, and the matrix is not invertible.

Now we have the collapse-detector. Point it at $A - \lambda I$.

The characteristic equation

We need $\det(A - \lambda I) = 0$. Compute the determinant for $A = \begin{pmatrix}a&b\\c&d\end{pmatrix}$:

$$\det\begin{pmatrix}a-\lambda&b\\c&d-\lambda\end{pmatrix} = (a-\lambda)(d-\lambda) - bc = \lambda^2 - (a+d)\lambda + (ad-bc)$$

The two coefficients have standard names. The sum of diagonal entries $a+d$ is the trace, written $\operatorname{tr}(A)$. The expression $ad-bc$ is the determinant, $\det(A)$. Setting the whole expression to zero gives the characteristic equation:

$$\lambda^2 - \operatorname{tr}(A)\,\lambda + \det(A) = 0$$

Solve with the quadratic formula:

$$\lambda = \frac{\operatorname{tr}(A) \pm \sqrt{\,\operatorname{tr}(A)^2 - 4\det(A)}}{2}$$

Two eigenvalues, in general. They're real only when the discriminant $\operatorname{tr}^2 - 4\det \geq 0$. That's why the rotate-90° matrix has none: its trace is $0$, determinant is $1$, discriminant is $-4$.

Once you have an eigenvalue $\lambda$, substitute it into the top row of $(A - \lambda I)\mathbf{v} = \mathbf{0}$. Write $\mathbf{v} = (x, y)^\top$: the equation is $(a-\lambda)x + by = 0$, giving $y = \frac{\lambda - a}{b}\,x$. The bottom row gives the same constraint — it must, because choosing $\lambda$ to make $\det(A-\lambda I)=0$ forces the rows to be proportional. Pick $x = b$ and the eigenvector is $(b,\,\lambda - a)$. Any scalar multiple also works. The eigenvalue defines a line, not a single arrow — an eigenspace.

Verification. For $A = \begin{pmatrix}2&1\\1&2\end{pmatrix}$: $\operatorname{tr} = 4$, $\det = 3$, so $\lambda^2 - 4\lambda + 3 = (\lambda-1)(\lambda-3) = 0$, giving $\lambda_1 = 3,\,\lambda_2 = 1$. Eigenvectors: $\mathbf{v}_1 = (1, 3-2) = (1,1)$ and $\mathbf{v}_2 = (1, 1-2) = (1,-1)$. Check: $\begin{pmatrix}2&1\\1&2\end{pmatrix}\begin{pmatrix}1\\1\end{pmatrix} = \begin{pmatrix}3\\3\end{pmatrix} = 3\begin{pmatrix}1\\1\end{pmatrix}$ ✓

There's one more thing worth feeling before we use this: what happens when you apply the same matrix again and again.

Repetition reveals the dominant direction

Write any starting vector as a blend of the two eigenvectors: $\mathbf{v}_0 = \alpha\,\mathbf{e}_1 + \beta\,\mathbf{e}_2$. Apply $A$ once: each component is multiplied by its eigenvalue. Apply it $k$ times:

$$A^k\mathbf{v}_0 = \alpha\lambda_1^k\,\mathbf{e}_1 + \beta\lambda_2^k\,\mathbf{e}_2$$

If $|\lambda_1| > |\lambda_2|$, then $\lambda_1^k$ grows faster. The $\mathbf{e}_1$ component dominates, and the direction of $A^k\mathbf{v}_0$ swings toward $\mathbf{e}_1$, locking on. The eigenvector with the largest absolute eigenvalue is magnetic. This is called power iteration.

That "direction everything piles onto" is exactly what we need to find the grain of real data.

The hidden axis in fish data

Measure 30 fish: length in centimetres and weight in arbitrary units. Plot each fish as a point $(L, W)$ and you get a tilted blob — bigger fish tend to be both longer and heavier, so the cloud has a grain. The question is: what single direction best captures how these fish vary?

Build a matrix that captures the shape of the cloud. First, centre the data by subtracting the mean of each measurement. Then form the spread matrix — formally the covariance matrix:

$$C = \begin{pmatrix}\operatorname{var}(L) & \operatorname{cov}(L,W) \\ \operatorname{cov}(L,W) & \operatorname{var}(W)\end{pmatrix}$$

Each entry has a plain meaning: $\operatorname{var}(L)$ is how much lengths vary across fish; $\operatorname{var}(W)$ is how much weights vary; $\operatorname{cov}(L,W)$ is how much they vary together — large and positive here, because longer fish are consistently heavier. $C$ is symmetric, so its eigenvectors are always real and perpendicular.

Now invoke the power-iteration insight. $C$ stretches space most along the direction of the cloud's grain. Repeatedly apply $C$ to any vector and it piles onto that grain direction — the dominant eigenvector $\mathbf{e}_1$. The eigenvalue $\lambda_1$ tells you how much of the total spread lives there: $\lambda_1/(\lambda_1+\lambda_2)\times 100\%$.

Once the axis is known, each fish gets a size score — its projection onto $\mathbf{e}_1$. Two fish with matching scores are the same size archetype: compatible. A fish's perpendicular distance from the axis is its "unusualness": how much it deviates from the natural weight-for-length relationship.

The axis was never labelled in the data. No one encoded it. But it was the dominant eigenvector all along — forced by the same three axioms we started with.

The chain, closed

We started from the definition of a coordinate system. A matrix relocates the reference arrows; every vector follows. A few special directions — eigenvectors — are only stretched, never rotated: $A\mathbf{v} = \lambda\mathbf{v}$. To find them, we rearranged the equation into a collapse condition, then built the determinant from three axioms (multilinear, alternating, normalised), which forced the formula $ad - bc$ and its geometric meaning as a signed area scale factor. The resulting characteristic equation yielded the eigenvalues; back-substitution gave the eigenspaces; power iteration explained why the dominant eigenvector emerges from repeated application. The fish data closed the loop: a hidden axis, invisible in the raw measurements, revealed as the dominant eigenvector of the spread matrix.

The technique of ranking all eigenvectors of the spread matrix by eigenvalue — keeping the top $k$ to compress data while retaining as much variation as possible — is called Principal Component Analysis (PCA). That's where this chain leads next.
